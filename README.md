# Outscale SDK for Golang

Welcome to Outscale SDK for [golang](https://golang.org/).

## How to use the SDK ?

Simply grab go module:
```bash
$ go get github.com/outscale-dev/osc-sdk-go/osc
```

See [examples](examples/) folder to jump straight into the code!
